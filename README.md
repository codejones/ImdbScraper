# ImdbScraper
Scrapy based imdb scrapper


## Spiders:


  top250    - Scrapes top 250 movies form Imdb
  
  
  top250tv  - Scrapes top 250 tv series from Imdb
 

## Dependencies:


  Python 2.7
  
  
  Scrapy Framework
  
  
## Usage:
 
 
 -scrapy crawl top250 -o out_movie.csv 
 
 
 -scrapy crawl top250tv -o out_tv.csv
